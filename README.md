# delivery-data-analysis
Data analysis of delivery orders using Python (cost, distance, and performance metrics).

# Delivery Orders Analysis (Python)

This project is a simple data analysis script built in Python to process and analyze delivery order data. It focuses on extracting useful insights such as total costs, delivery times, and city-based performance metrics.

# Features

Filter orders based on distance
Calculate total cost for specific cities (e.g., Bogotá)
Compute average delivery time
Aggregate total costs by city
Identify the city with the highest total cost

# Technologies Used

Python (no external libraries required)
Basic data structures (lists, dictionaries)
Control flow and functions
